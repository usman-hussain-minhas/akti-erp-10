## Schema Decision

The existing `EventOutbox` table already owned the durable outbox row. P5B-017b extended that model instead of creating a duplicate event-envelope table.

Added envelope support:

- `event_id`
- `producer`
- `occurred_at`
- `schema_version`
- `source_module`
- `subject`
- `privacy_class`
- `retention_class`
- `redaction_policy`
- `audit_required`
- `replay_allowed`

The migration is additive. Existing rows keep nullable/defaulted envelope fields, and new writes populate the fields from the P5B-017a validator.
