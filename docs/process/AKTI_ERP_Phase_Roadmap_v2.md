## Module Classification

- Lead Desk is a business module, not a core platform module.
- Engagement Gateway Lite is a shared platform module.
- WhatsApp stub is an integration adapter and remains non-production.

## Planning Rule

Do not implement future phases from this roadmap summary. Create phase-specific planning and control docs from the current repo state before implementation. Phase 4A/4B ticket packs require separate approval.
